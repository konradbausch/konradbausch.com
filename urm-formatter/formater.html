URM Formatter

This tool simplifies the creation of URM Files for groups.

If you possess a configuration file, simply upload it.

Otherwise, select "New Config," specify the number of teams, add students to these teams, and remember to download your configuration for future use. Next, allocate points to each team, input the assignment number, and set the maximum points.

Once completed, download the generated URM File.
